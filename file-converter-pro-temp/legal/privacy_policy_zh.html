File Converter Pro 隐私政策

最后更新：2026年2月

1. 简介

我们非常重视您个人数据的保护。本隐私政策说明了 File Converter Pro 如何处理您的信息。

2. 隐私理念

File Converter Pro 是一款 100% 离线应用程序。我们坚信您的文件属于您本人，应当保留在您的设备上。

3. 信息收集

3.1 个人数据

我们不收集任何个人数据。有关您的任何信息均不会发送至我们的服务器或第三方。

3.2 文件处理

您转换、合并、压缩或修改的所有文件均在您的本地计算机上处理。任何文件都不会通过互联网传输。

4. 文件使用

您的文件：

  • ✅ 仅在您的设备上处理 — 不会传输至外部服务器
  • ✅ 不会共享给第三方 — 我们从不分享您的文件
  • ✅ 按您的指示删除 — 您对自己的数据拥有完全控制权
  • ✅ 必要时临时存储 — 仅用于正在进行的操作

5. 本地技术数据

5.1 配置文件

应用程序会将您的偏好设置（主题、语言、设置）以加密配置文件的形式保存在您的本地设备上。

5.2 转换历史记录

为方便起见，转换历史记录将在您的设备本地存储。您可以随时通过设置将其删除。

5.3 使用统计

部分匿名统计数据（如转换次数、使用的格式）可能会在本地收集，以改善用户体验。这些数据仅保留在您的设备上。

6. 信息共享

任何信息均不会与第三方共享。我们不会向任何公司或组织出售、出租或共享您的数据。

7. 安全

我们实施了强有力的安全措施来保护您的文件：

  • 🔒 本地加密敏感配置文件
  • 🔒 进程隔离以防止数据泄露
  • 🔒 使用后自动清理临时文件
  • 🔒 最小权限 — 应用程序仅访问您选择的文件

8. Cookie 及类似技术

本应用程序不使用 Cookie。作为离线桌面应用程序，不使用任何 Cookie 或网络追踪技术。

9. 您的权利

作为用户，您有权：

  • 👁️ 访问所有本地存储的数据
  • ✏️ 随时修改您的偏好设置
  • 🗑️ 完全删除所有应用数据
  • ⚙️ 导出您的设置和历史记录

10. 同意

使用 File Converter Pro 即表示您同意本隐私政策，并确认：

  • ✅ 所有处理均在本地完成
  • ✅ 不收集或发送任何数据
  • ✅ 您负责备份自己的文件

11. 政策修改

我们可能会更新本隐私政策。任何重大变更将在应用程序启动时进行通知。
